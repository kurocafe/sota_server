# Sota Server
研究で使用している音声対話ロボット🤖（Sota）の主な機能がある。

## 概要（主な機能）
- 音声認識（音声からテキストに変換）
- 文章生成（テキストを入れて、それに対する返答）
- 音声合成（生成された文章を音声に変換）
- QRコードの生成（ユーザー認識するため）
- QRコードを読み取る
